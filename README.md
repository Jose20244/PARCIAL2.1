El sistema permite al usuario ingresar números enteros de forma dinámica, los cuales son procesados y clasificados automáticamente en diferentes estructuras según su valor.
Además, el programa mantiene un historial de operaciones, permite la construcción de un árbol binario de búsqueda (BST), realiza operaciones de eliminación y reorganización de datos,
y muestra estadísticas relevantes.
El programa funciona mediante un menú interactivo que permite al usuario acceder a todas las funcionalidades de manera organizada.
Se utilizo Punteros y memoria Dinamica, Estructuras, Lista enlazada, pila, cola, lista doblamente enlazada, Arbol (BST).

Cada número ingresado es evaluado:

Si es par y positivo se inserta en la cola.
Si es impar y positivo se inserta en la pila.
Si es negativo se inserta en la lista simple.
Si es cero no se almacena y se registra como inválido.
